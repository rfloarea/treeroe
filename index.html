Questions from the Audience

How is this site made?

Thanks for asking! It's super simple:

  1. Registered my domain name through Porkbun
  2. Static hosting through Porkbun
  3. Used Porkbun's feature that syncs to a GitHub repo
  4. Created a GitHub repo for this site and clicked some buttons to connect them
  5. And then, ya know, typed this stuff
